# Internship-Skill-Evaluation-Platform.

A full-stack web application built to evaluate internship skills through user registration, role-based login, task submission, and certificate generation.

Tech Stack

Frontend: React.js, HTML, CSS

Backend: Node.js, Express.js

Database: MySQL

Features

User registration & login

Role-based access (Admin / User)

Task / form submission

Task completion tracking

Certificate generation (PDF)

MySQL database integration
